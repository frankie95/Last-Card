========================================================================
    APPLICATION : LastCard Project Overview
========================================================================

This file contains a summary of what you will find in each of the files that
make up your LastCard application.

lastcardlib
    The contain the implementation of card, player and operationofgame class. All
    function can be see in the lastcardlib.cpp file, and all header in the lastcard.h file.

loginlib
    The contain the implementation of login class. All function can be see in the 
    loginlib.cpp file, and all header in the login.h file.

lastCard
    This is the main application source file.
    Contains the code to display the form.

/////////////////////////////////////////////////////////////////////////////
Other standard files:

<stdlib.h>, <time.h>, <fstream>,<string.h>
    These files are used to build a precompiled header (PCH) file
    named LastCard.pch and a precompiled types file named StdAfx.obj.

/////////////////////////////////////////////////////////////////////////////

To Run the program, please follow the following steps.
1 - Open LastCard
2 - Change directory to LastCard
3 - Run LastCard.sln
4 - Press F7
5 - Press CTRL+F5
!!@Open the game in different computer will have different effect, we do recommand to rebuild the game@!!
!!@For best gaming experience, we recommanded the size of game.h not smaller than (1100,720)@!!
